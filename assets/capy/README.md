## 팁
- 카피바라를 화면 가운데에, 비슷한 크기/여백으로 (연못에 나란히 놓일 때 일관성)
- 투명 배경(transparent), 바닥 장식은 최소화하면 물 위에 깔끔하게 올라갑니다
